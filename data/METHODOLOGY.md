# Lookuptable Mapping Methodology

## Purpose

Map every word in `words.json` (370,105 English words) to one of the 72 ancient roots
defined in `root_words.json`. Each word is honored by its **original meaning** — the
meaning the word carried historically, not modern reinterpretation.

## Mapping Strategy (3 Layers)

### Layer 1: Manual Base Dictionary
Hand-curated mappings of common English base words to their true root.
Each mapping includes the word's honored original meaning.
This layer covers ~3,000+ high-frequency base words.

**Principle:** Honor the word's own meaning. Look at what the word actually
meant when it was first used, not what people today think it means.

**Example:**
- "woke" → root "ir" — "became alert, opened the eyes" (not a political label)
- "sad" → root "sat" — "sated, full, weary from excess" (not merely unhappy)
- "nice" → root "rn" — "precise, exact, fine in distinction" (from Latin nescius: ignorant, then precise)

### Layer 2: Morphological Derivation
For words not in the base dictionary, strip affixes to find the base form:

**Suffixes stripped (in order):**
- Inflectional: -s, -es, -ed, -d, -ing, -er, -est
- Derivational: -ly, -ness, -ment, -tion, -sion, -ity, -ance, -ence,
  -ful, -less, -ous, -ive, -able, -ible, -al, -ial, -ize, -ise, -ify,
  -ate, -let, -ling, -ish, -ic, -ist, -ism, -dom, -ship, -ward, -wise,
  -like, -ory, -ary, -ery, -ure, -ous, -eous, -ious

**Prefixes stripped:**
- un-, re-, pre-, dis-, mis-, over-, under-, out-, non-, anti-, counter-,
  super-, sub-, inter-, trans-, de-, en-, em-, in-, im-, ir-, il-

**Rule:** A derived word inherits the root of its base word.
"walking" → base "walk" → root "pa" (path, opening the way)

### Fallback: Letter-Pattern Heuristics
For truly unresolvable words (rare/technical/borrowed), assign based on
the dominant etymological pattern of words sharing similar structure.

## Consistency Rules

1. **Same base, same root.** All forms of a word share one root.
2. **Meaning over sound.** Map by true meaning, not surface similarity.
3. **Oldest meaning wins.** When a word's meaning shifted, honor the original.
4. **One root per word.** Each word gets exactly one root (the primary one).
5. **Compound words** get the root of the head word (the main meaning-carrier).

## Generator Script

The file `src/generate_lookuptable.js` implements this methodology.
Run it with: `node src/generate_lookuptable.js`

It processes all 370K words and outputs `data/lookuptable.json`.
