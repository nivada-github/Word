# Word Vortex

## Run locally

```bash
npm install
npm start
```

The server starts on `http://localhost:3000` (override with the `PORT` env var).

## Development

```bash
npm run dev    # start with --watch (auto-restart on changes)
npm test       # run tests
```

## Tech Stack

- **Express** -- web framework
- **cors** -- cross-origin support
- **Jest + Supertest** -- testing
